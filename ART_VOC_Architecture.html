üîÑ Data Flow Pipeline

Step 1: Ingestion

JSON files uploaded to Unity Catalog Volumes. Auto Loader detects new files and triggers pipeline.

Step 2: Bronze Processing

DLT streaming tables ingest raw data. Checkpoint tracks processed files. 68 interactions loaded.

Step 3: Silver Enrichment

NLP sentiment analysis applied. Topics extracted. Data quality checks performed. 5 Positive, 56 Neutral, 7 Negative.

Step 4: Gold Aggregation

Member 360 views created. Churn risk calculated. Behavioral segments assigned. 50 unique members.

Step 5: ML Training

XGBoost model trained on 5K synthetic samples. 9 features used. Model registered in MLflow as Production.

Step 6: Batch Inference

ML predictions applied to all Gold table members. Churn probability scores generated. Risk levels assigned.

Step 7: Visualization

Executive dashboard queries ml_churn_predictions table. Real-time metrics displayed. High-risk members prioritized.

Step 8: Action

Stakeholders view dashboard. Identify at-risk members. Trigger retention campaigns. Monitor effectiveness.
